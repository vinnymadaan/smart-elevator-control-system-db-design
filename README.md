# 🛗 Smart Elevator Control System — ER Diagram

## 📌 Overview

This project presents the database design (ER Diagram) for a **smart elevator control platform** used in large-scale infrastructures like corporate buildings, malls, airports, hospitals, and high-rise residential complexes.

The system is designed to manage **multiple buildings**, each containing several elevators operating across multiple floors, handling thousands of ride requests daily.
